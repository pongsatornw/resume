# GitHub-style software engineer résumé

A responsive, dependency-free résumé template inspired by GitHub's repository interface.

## Use locally

Open `index.html` in a browser. Edit the sample profile, experience, projects, links, and skills directly in that file.

## Export to PDF

Click **Export PDF**, then select **Save as PDF** in the browser print dialog. The print stylesheet uses an A4 page layout and removes the repository navigation and controls automatically.
